# RecommendedCommunitySim
Modeling how a music listening community grows under recommendation algorithms with different incentives. 

See details of the Game, each agent type, recommenders, etc. in `writeup.pdf`.

Please take a moment to look at `/results/` to see **Community Graph Evolution, Agent Action Log, and Individual Agent Statistics**! 

# Use Guide:
*Requirements*:
```
numpy
matplotlib
networkx
```

Run `python music_rec_sim.py` to run the model simulation. Hyperparameters for agents, number of each type of agent, type of recommender, etc. can be changed in `main()` of `music_rec_sim.py`.
